# lrj2u
UEX static pages

## 后台设置

本项目设置页面的 `GitHub Pages` 区域内，Custom domain 输入 uex.com，然后保存。

## 域名配置

两个方法，先尝试第一个（这个比较干净）

首先将 uex.com 域名 CNAME 记录建立两个，指向 github 的地址：

`uex.com` -> lrj2u.github.io
`www.uex.com` -> lrj2u.github.io

第二个方式是将 uex.com 域名的 A 地址指向（这个我用过）：

```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```
